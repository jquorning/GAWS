# GAWS - General Ada Web Server

## Status
Initial

## Dependencies

* Aws - Ada Web Server
* DK8543.Text.Comments
* Simple_Components - For SQLite3 and SQLite3 binding

## Configuring
```sh
ed var/PROGRAM_NAME
ed var/PROGRAM_VERSION
make setup
```

## Building
```sh
make
```
or
```sh
gprbuild
```

## Running
```sh
run-gaws
```
