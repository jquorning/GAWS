GAWS - General Ada Web Server

GAWS is (going to be) a Web Server like Apache and IIS.

This page is bundled with the GAWS sources.

GAWS has no homepage of its own for now.

Features

  • Static page server
  • Can handle a lot of small pages with not much traffic
  • Multi-threaded
  • Easy configuration

    State

    GAWN is today (2019-02-22) in usable state

    Future

  • Simple Contents Management System
